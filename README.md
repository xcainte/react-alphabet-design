# react-alphabet-design
Design concept of React components

## Types of components

### `block`
block components

### `container`
container components

### `dialog`
dialog components

### `fragment`
components that make up part of a page

### `grid`
grid list components

### `item`
item components

### `list`
list components

### `modal`
modal components

### `page`
page components

### `root`
Top-level components

### `ui`
UI components


## Component level

- Ground
    - `root` > `page` > `fragment` > `container` > `block` > `grid`, `list` > `item` > `ui`
- Floating
    - `dialog`, `modal`
